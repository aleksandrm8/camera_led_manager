# CAMERA LED MANAGER
Client/server application for camera LED management.
## DEPENDENCIES
g++ 4.8.4

GNU make 3.81

tput

stty
## GETTING STARTED
To build and run server, cd to programm directory and run:

    $ make && ./server
    
To run client run from root directory:

    $ ./client.sh
    
You may run one ore more client programm.
